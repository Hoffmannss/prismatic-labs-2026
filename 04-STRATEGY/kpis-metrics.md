## RED FLAGS TO MONITOR

| Warning Sign | Action |
|-------------|--------|
| **Conversion <3%** | Revisit messaging/positioning |
| **Churn >10%** | Investigate satisfaction drop |
| **CAC >R$1.000** | Optimize marketing channels |
| **NPS <40** | Quality audit + client feedback |
| **Delivery delay >25 days** | Process improvement |
| **Revision requests >3/project** | Design process tweak |
